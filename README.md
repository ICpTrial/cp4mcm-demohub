# IBM Cloud Pak for Multicloud Management ハンズオン

IBM Cloud Pak for Mutlicloud Managemnet(以下 CP4MCM)を利用した マルチクラスター環境のハンズオンです。  
この ハンズオン環境を利用することで、 複数 Kubernetesクラスターを対象としたマルチクラウド環境に対する操作を 体験することが可能です。
